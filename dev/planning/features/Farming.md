# 개요

---

[[VanillaFarming]]를 기반으로 확장된 농사 콘텐츠 입니다.

바닐라 농사에서 재배가 가능한 작물들은 다계절에 영향을 받지 않고 [농사 기획](https://www.notion.so/25cd1c757eee81409321c0a97c5dab25?pvs=21) 에서 재배되는 작물들만 다계절에 영향을 받습니다.

한 계절은 총 28일이며 마인크래프트 시간으로 계산 되면 1일마다 계절이 변경 됩니다.

계절이 바뀌면 다계절 작물을 제외하고 이전까지 키우던 작물들은 모두 시들어 버립니다.

게임 초반에는 [[초반 퀘스트]]를 노리면서 키우는 게 좋습니다.

# 아이디어 기획

## 핵심콘텐츠 1

- 농작물: 밀, 당근, 감자, 사탕무 ...
- 필수 인프라: 경작지, 물 공급, 광원, 상자(농작물 저장할 것)
- 준비물: 호미 1개, 씨앗, 물양동이 -> CustomCrops plugin에 물뿌리개 도구가 있던데, 스타듀밸리처럼 물 뿌리고 몇 시간 지나면 경작지가 마르는 알고리즘인지 알아봐야 할듯

## 핵심시스템 1

- 호미 티어별 성능: 내구도 차이만 있음, 작업 속도는 동일함
    - 호미를 하나로 통일하고, 내구도를 강화하는 방법으로 바꿔야 할듯
    - 즉 호미의 내구도 = 호미의 레벨
    - 내구도를 다 사용한다고 해서 호미가 박살나는 게 아니라, 사용을 못함
    - 농작물을 수집할 때에는 무조건 호미를 사용하여 (내구도 사용을 극대화 시키기)
        - 즉 농작물의 grow level 별로 내구도를 먹는 속도가 다름
        - 이 때, 호미의 내구도 리미트를 걸어야 됨 (재밌는 다른 부가요소가 없는지도 파악해봐야 할듯)
    - 농사를 짓고, 작물을 수확할 때마다 #농부 레벨이 올라감
    - 농부 레벨이 올라가면 올라갈수록 농작물 수확 효율, 행운, (벌레 드롭?), n칸 농작물 수확 스킬 등을 해금할 수 있음
        - 농부 레벨을 올려 다음 #농부스킬트리 를 해금하려면 농작물을 캐면 나오는 crop coin이 있어야만 해금할 수 있음
        - 각 트리별로 해금해야하는 갯수가 정해져 있음

## 스타듀밸리의 농장 시스템

스타듀밸리의 농장 시스템은 1칸당 1개의 작물을 심을 수 있습니다. 물뿌리개로 물을 주어 농작물이 자랄 수 있도록 도와줍니다.
