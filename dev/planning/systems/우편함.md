기본 명령어
```
/mail open <player> - player의 메일함을 볼 수 있습니다.
/mail send <player> - player에게 상품을 보냅니다 (한개 또는 64개만) (GUI가 열리면 가운데에 아이템을 넣어서 버튼(보내기/취소)를 누를 수 있음.)
/mail edit <player> - player의 메일함을 수정할 수 있습니다. (log 기록 남음)
/mail reload - 비동기 DB와 redis로 메일함을 갱신합니다.

# 이벤트
플레이어가 접속 했을 때, 메일함에 아이템이 있다면 {알림 메세지}를 전송함
```
