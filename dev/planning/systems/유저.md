# user sql 설계
```
username: string
uuid: string
create_ip: string
last_update_ip: string
locked: date
rank_expired: date
date_created: date
last_updated: date
enabled: boolean
```

```java
public interface User {  
    UUID getUuid();  
    String getUsername();  
    String getCreateIp();  
    String getLastUpdateIp();  
    void setLastUpdateIp(String ip);  
  
    Date getLockedDate();  
    void setLockedDate(Date date);  
    boolean isLocked(); // 비즈니스 로직: 현재 시간 기준 잠김 여부 확인  
  
    Date getRankExpiredDate();  
    void setRankExpiredDate(Date date);  
  
    Date getDateCreated();  
    Date getLastUpdated();  
    void setLastUpdated(Date date);  
  
    boolean isEnabled();  
    void setEnabled(boolean enabled);  
}
```
설계도를 먼저 만들어줍니다.
```java
public interface UserStrategy {  
    CompletableFuture<User> getUser(UUID uuid);  
    CompletableFuture<User> createUser(UUID uuid, String username, String ip);  
    CompletableFuture<Void> saveUser(User user);  
}
```
만들어둔 설계도를 바탕으로 CompletableFuture로 비동기로 데이터를 불러오고 저장하고, 생성할 수 있도록 합니다.
